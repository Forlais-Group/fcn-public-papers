# v1.0-public

Status: upload-ready.

First public release of the Forlais Compression Notation BF16 trained-weight compression paper pack.

## Contents

This release contains nine selected papers covering:

- Shannon source-coding framing for trained neural network weights
- the BF16 single-model 1.52x entropy ceiling
- Benford's Law as the origin of the BF16 compression ceiling
- the mutual-information paradox in BF16 weight streams
- sign, exponent, and mantissa component decomposition
- the r = 2 law of mantissa-bit bias
- a negative result for r = 2 prior seeding in adaptive coding
- a public lossless-compressor comparison on BF16 model weights
- aligned byte compression as a simple public-safe transform baseline

## Release Boundary

This release does not include non-public source logs, operational code, model weights, non-public datasets, hidden benchmark mechanics, operational procedures, C04 cross-model delta codec stack details, non-public deduplication papers, or traversal strategy papers.
